## calculate value of roman numeral

```js
var values = {
	I: 1,
	V: 5,
	X: 10,
	L: 50,
	C: 100,
	M: 1000
};
```


numeral | value
------- | -----
VI  | 6
IV | 4
CVL | 145
XIII | 13
CMIL | 949

multiple preceding decrementors not valid, i.e. `IIV`
